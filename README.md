🚀 AI Website Generator


-An AI-powered web application that generates complete websites using user input.

-Built using Node.js, Express, and OpenAI API.





📌 Features



-🤖 AI-generated website content

-🎨 Dynamic website preview

-🔐 Secure backend API handling

-⚡ Fast and responsive UI

-🌐 REST API integration






🛠️ Tech Stack



-Frontend: HTML, CSS, JavaScript

-Backend: Node.js, Express.js

-AI Model: OpenAI API

-Other Tools: dotenv, cors
